# AI Object Detection System

This project is developed as part of the **Artificial Intelligence Internship at CodeAlpha**.

## 📌 Project Description
This project demonstrates real-time object detection using a pre-trained YOLO model and OpenCV. The system captures video from the webcam and detects objects in real time.

## 🚀 Features
- Real-time webcam detection
- Detect multiple objects
- Bounding boxes around detected objects
- Fast and accurate object recognition

## 🛠 Technologies Used
- Python
- OpenCV
- YOLOv8
- Ultralytics
